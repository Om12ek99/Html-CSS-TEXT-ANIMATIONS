Parliamo delle transizioni

E’ una proprietà CSS abbreviata che permette di dare degli effetti dinamici
agli elementi della pagina, ed è composta da 4 proprietà:


  • Transition-property

    Specifica il nome della proprietà CSS a cui è rivolto l’effetto di transizione (es: color, width…)

  • Transition-duration

    Specifica quanti secondi o millisecondi sono necessari per completare un effetto di transizione.

  • Transition-timing-function

    Specifica la curva di velocità dell’effetto di transizione (es: ease, linear…)

  • Transition-delay

    Specifica dopo quanto tempo deve partire l’effetto di transizione.
